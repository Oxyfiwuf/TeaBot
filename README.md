The project was implemented to record and view statistics of tips earned in restaurants
The aiogram library is used to connect to Telegram, and openpyxl to import and view user records

To record, use /start -> "Додати / змінити суму" -> Select a date from the proposed ones or enter manually and add the amount
To view, use /start -> "Статистика" -> Select a period or enter manually
The administrator has the /a function available to export data to Excel and /users to view users who have launched the bot
